## 环节三：任务驱动教学

### 任务一：工业制取SO₂的原料与反应
*   **[自主学习]** 阅读教材中关于工业制硫酸原料的部分，找出制取SO₂的主要原料及其反应方程式。思考：为什么选择这些原料？
*   **归纳小结**：
    *   主要原料：硫磺（S）或硫铁矿（FeS₂）。
    *   反应方程式：
        *   S(s) + O₂(g) $\xrightarrow{点燃}$ SO₂(g)
        *   4FeS₂(s) + 11O₂(g) $\xrightarrow{高温}$ 2Fe₂O₃(s) + 8SO₂(g)
    *   硫铁矿是更常见的原料，其燃烧产物Fe₂O₃可用于炼铁，实现资源综合利用。
*   **评价训练**：
    1.  下列关于工业制硫酸原料的说法不正确的是（ ）
        A. 硫磺是制取SO₂的常见原料之一。
        B. 硫铁矿燃烧时，铁元素被氧化。
        C. 硫铁矿燃烧的产物Fe₂O₃可用于炼铁。
        D. 硫铁矿燃烧时，硫元素被还原。
        *   （答案：D）
    2.  写出硫铁矿在空气中燃烧生成二氧化硫的化学方程式。
        *   （答案：4FeS₂(s) + 11O₂(g) $\xrightarrow{高温}$ 2Fe₂O₃(s) + 8SO₂(g)）

### 任务二：二氧化硫的物理性质与酸性氧化物性质
*   **[合作探究]** 观察SO₂的物理性质（颜色、状态、气味、密度、溶解性），并结合已学知识，讨论SO₂作为酸性氧化物的性质。
    *   提示：SO₂能与水、碱反应。
*   **归纳小结**：
    *   **物理性质**：无色、有刺激性气味、有毒气体，密度比空气大，易溶于水。
    *   **化学性质（酸性氧化物）**：
        *   与水反应：SO₂(g) + H₂O(l) ⇌ H₂SO₃(aq) (亚硫酸)
        *   与碱反应：SO₂(g) + 2NaOH(aq) → Na₂SO₃(aq) + H₂O(l)
        *   与碱性氧化物反应：SO₂(g) + CaO(s) → CaSO₃(s)
*   **评价训练**：
    1.  下列关于SO₂的说法正确的是（ ）
        A. SO₂是一种无色无味的气体。
        B. SO₂溶于水后形成硫酸。
        C. SO₂能与NaOH溶液反应。
        D. SO₂的密度比空气小。
        *   （答案：C）
    2.  将SO₂通入澄清石灰水中，现象是_______，写出反应的化学方程式。
        *   （答案：先变浑浊，后又变澄清；SO₂(g) + Ca(OH)₂(aq) → CaSO₃(s)↓ + H₂O(l)；CaSO₃(s) + SO₂(g) + H₂O(l) → Ca(HSO₃)₂(aq)）
