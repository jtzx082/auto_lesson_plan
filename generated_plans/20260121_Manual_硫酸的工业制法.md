### 任务一：探究接触法制硫酸的反应原理及条件优化

**1. 合作探究**
*   **问题设计：** 接触法制硫酸的核心反应是2SO₂(g) + O₂(g) ⇌ 2SO₃(g) ΔH < 0。请同学们结合化学平衡移动原理、反应速率理论和工业生产实际，讨论以下问题：
    1.  为了提高SO₂的转化率，应选择怎样的温度、压强和SO₂与O₂的投料比？
    2.  工业上实际选择的反应条件（如400~500℃、常压或微压、过量O₂）与理论分析是否存在差异？为什么？
    3.  选择V₂O₅作为催化剂的目的是什么？它对SO₂的转化率有影响吗？
*   **学习活动：** 学生分组讨论，查阅资料（教材、网络），代表汇报，教师引导和补充。

**2. 归纳小结**
*   **反应原理：** 2SO₂(g) + O₂(g) ⇌ 2SO₃(g) ΔH < 0，是一个可逆的放热反应，且正反应是气体体积减小的反应。
*   **条件选择：**
    *   **催化剂：** V₂O₅（五氧化二钒），提高反应速率，不改变平衡，但能使SO₂在较低温度下达到较高的转化率。
    *   **温度：** 400~500℃。理论上低温有利于平衡正向移动，但温度过低反应速率慢。此温度是综合考虑反应速率、转化率和催化剂活性后的最佳选择。
    *   **压强：** 常压或微压。理论上加压有利于平衡正向移动，但加压设备成本高，且对转化率提高不显著，故工业上多采用常压或微压。
    *   **投料比：** 增大O₂的浓度（过量O₂），有利于提高SO₂的转化率，同时降低SO₂的排放。

**3. 评价训练**

*   **学习活动：** 学生分组阅读教材流程图，讨论各环节目的和原理，完成学习单，代表汇报，教师点评。

**2. 归纳小结**
*   **生产流程：**
    1.  **造气（沸腾炉）：** 燃烧硫磺或黄铁矿（FeS₂），生成SO₂。
        *   S + O₂ → SO₂
        *   4FeS₂ + 11O₂ → 2Fe₂O₃ + 8SO₂
    2.  **净化（净化塔、洗涤塔、干燥塔）：** 除去SO₂气体中的粉尘、水蒸气及其他杂质，防止催化剂中毒，提高转化率。
        *   洗涤：水或稀H₂SO₄
        *   干燥：浓H₂SO₄
    3.  **接触氧化（接触室）：** SO₂在V₂O₅催化剂作用下氧化生成SO₃。
        *   2SO₂(g) + O₂(g) ⇌ 2SO₃(g)
    4.  **吸收（吸收塔）：** SO₃用98.3%的浓H₂SO₄吸收，生成发烟硫酸，再加水稀释得到浓H₂SO₄。
        *   SO₃ + H₂SO₄ → H₂S₂O₇（发烟硫酸）
        *   H₂S₂O₇ + H₂O → 2H₂SO₄
*   **关键点：** 净化是保证催化剂活性的关键；浓H₂SO₄吸收SO₃是为了避免形成酸雾，提高吸收效率。

**2. 归纳小结**
*   **绿色化学理念应用：**
    *   提高SO₂的转化率（如采用过量O₂、优化反应条件）。
    *   对废热进行回收利用，节约能源。
*   **SO₂尾气处理：**
    *   **碱液吸收法：** 用NaOH溶液或Ca(OH)₂悬浊液吸收。
        *   SO₂ + 2NaOH → Na₂SO₃ + H₂O
        *   SO₂ + Ca(OH)₂ → CaSO₃ + H₂O
    *   **石灰石-石膏法：** 用石灰石浆液吸收SO₂，生成CaSO₃，再氧化生成CaSO₄·2H₂O（石膏）。
        *   SO₂ + CaCO₃ → CaSO₃ + CO₂
        *   2CaSO₃ + O₂ + 4H₂O → 2CaSO₄·2H₂O
*   **其他废弃物处理：** 沸腾炉产生的废渣（Fe₂O₃）可用于炼铁；废水经处理达标后排放。

## 环节四：课堂小结

**《硫酸的工业制法》知识体系**

**一、核心反应原理**
*   **造气：** S + O₂ → SO₂ 或 4FeS₂ + 11O₂ → 2Fe₂O₃ + 8SO₂
*   **接触氧化：** 2SO₂(g) + O₂(g) ⇌ 2SO₃(g) ΔH < 0（可逆放热、气体体积减小）
*   **吸收：** SO₃ + H₂SO₄ → H₂S₂O₇；H₂S₂O₇ + H₂O → 2H₂SO₄

**二、生产流程与设备**
1.  **造气：** 沸腾炉（S或FeS₂燃烧）
2.  **净化：** 净化塔、洗涤塔、干燥塔（除尘、洗涤、干燥，防催化剂中毒）
3.  **接触氧化：** 接触室（V₂O₅催化剂，400~500℃，常压或微压，过量O₂）
4.  **吸收：** 吸收塔（98.3%浓H₂SO₄吸收SO₃，避免酸雾）

**三、条件选择原则**
*   **平衡移动原理：** 提高转化率
*   **反应速率理论：** 提高生产效率
*   **经济成本：** 设备、原料、能源
*   **环境保护：** 减少污染

**四、绿色化学与环境保护**
*   **提高转化率：** 减少SO₂排放
*   **尾气处理：** 碱液吸收法（如Ca(OH)₂、NaOH），石灰石-石膏法
*   **废渣利用：** Fe₂O₃炼铁
*   **废热利用：** 节约能源

**五、核心素养**
*   **宏观辨识与微观探析：** 理解工业生产与微观反应原理的联系。
*   **变化观念与平衡思想：** 运用化学平衡原理指导工业生产。
*   **证据推理与模型认知：** 通过流程图分析生产过程。
*   **科学探究与创新意识：** 优化生产条件，解决实际问题。
*   **科学精神与社会责任：** 关注环境保护，树立可持续发展理念。
