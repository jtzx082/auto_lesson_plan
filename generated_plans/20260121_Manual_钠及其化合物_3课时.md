# 第1课时：钠的物理性质与化学性质

## 环节一：学习目标
1. 了解钠的物理性质，认识钠是一种活泼的金属。
2. 掌握钠与氧气、水、酸等物质的反应，理解钠的化学活泼性。
3. 通过实验探究，培养观察、分析和归纳实验现象的能力。
4. 认识钠在生活中的应用及安全储存方法。

## 环节二：情景创设
* **情景类型**: 实验情景
* **内容**: 教师播放一段钠与水反应的视频，或进行小块钠投入水中的演示实验。引导学生观察现象（钠浮在水面，熔成小球，在水面游动，发出嘶嘶声，滴加酚酞后溶液变红）。
* **目的**: 提出问题：“钠为什么会剧烈反应？它有哪些独特的性质？”引发学生对钠的物理性质和化学性质的探究兴趣，明确本课时将学习钠的性质。
* **时间**: 3分钟

## 环节三：任务驱动教学

**任务一：探究钠的物理性质**
1. **[自主学习]**:
   * 教师展示一块保存在煤油中的钠块，引导学生观察其外观、颜色、光泽。
   * 学生查阅教材或资料，了解钠的硬度、密度、熔点、沸点、导电性、导热性等物理性质。
   * 思考：钠为什么要保存在煤油中？
2. **归纳小结**:
   * 钠是银白色、有金属光泽的固体，质软（可用小刀切割），密度比水小但比煤油大，熔点低（约97.8℃），是电和热的良导体。
   * 钠的保存：因其活泼性，通常保存在煤油或石蜡中，隔绝空气和水。
3. **评价训练**:
   * 1. 下列关于钠的物理性质的描述，不正确的是（ ）
      A. 银白色固体，有金属光泽
      B. 质地坚硬，不能用小刀切割
      C. 密度比水小，熔点低
      D. 是电和热的良导体
   * 2. 钠通常保存在煤油中，其主要目的是（ ）
      A. 防止钠挥发
      B. 防止钠与空气中的氧气和水反应
      C. 防止钠与煤油反应
      D. 降低钠的熔点

**任务二：探究钠与氧气的反应**
1. **[合作探究]**:
   * **实验探究**:
     * 现象一：将一小块新切开的钠暴露在空气中，观察其表面变化。
     * 现象二：取一小块钠，用滤纸吸干煤油，放在坩埚中加热，观察燃烧现象。
   * **讨论**:
     * 现象一说明了什么？写出反应方程式。
     * 现象二的产物是什么？与现象一的产物有何不同？写出反应方程式。
   * **思考**: 钠与氧气反应的条件和产物有何区别？
2. **归纳小结**:
   * 钠在常温下与氧气反应生成氧化钠（Na₂O）：4Na + O₂ → 2Na₂O (缓慢氧化，表面变暗)
   * 钠在加热或点燃条件下与氧气反应生成过氧化钠（Na₂O₂）：2Na + O₂ --(点燃)--> Na₂O₂ (剧烈燃烧，发出黄色火焰)
   * 钠具有很强的还原性。
3. **评价训练**:
   * 1. 将一小块钠暴露在空气中，下列现象的先后顺序是（ ）
      ① 表面变暗 ② 出现白色固体 ③ 形成白色晶体
      A. ①②③ B. ①③② C. ②①③ D. ③②①
   * 2. 钠在空气中燃烧时，火焰呈_______色，产物是_______（填化学式）。

**任务三：探究钠与水、酸的反应**
1. **[合作探究]**:
   * **实验演示**: 教师再次演示钠与水反应，并滴加酚酞溶液。
   * **观察与讨论**:
     * 观察钠的浮、熔、游、响、红现象，并解释其原因。
     * 酚酞变红说明了什么？
     * 写出钠与水反应的化学方程式。
   * **思考**: 钠与水反应的本质是什么？与酸反应呢？
2. **归纳小结**:
   * 钠与水剧烈反应，生成氢氧化钠和氢气：2Na + 2H₂O → 2NaOH + H₂↑
   * 现象解释：密度比水小（浮），熔点低（熔），反应放热使其熔成小球并在水面移动（游），生成氢气发出嘶嘶声（响），溶液呈碱性使酚酞变红（红）。
   * 钠与酸反应更加剧烈：2Na + 2HCl → 2NaCl + H₂↑ (由于反应剧烈且危险，一般不直接进行实验，钠会先与水反应)
3. **评价训练**:
   * 1. 钠与水反应的现象中，能说明钠的熔点低的是（ ）
      A. 钠浮在水面上
      B. 钠熔成小球
      C. 钠在水面游动
      D. 溶液变红
   * 2. 写出钠与水反应的化学方程式：____________________。

## 环节四：课堂小结
* **思维导图结构**:
  * **钠 (Na)**
    * **物理性质**
      * 银白色、金属光泽
      * 质软、密度小（比水）、熔点低
      * 导电、导热
    * **化学性质** (强还原性)
      * **与O₂**
        * 常温: 4Na + O₂ → 2Na₂O
        * 加热/点燃: 2Na + O₂ --(点燃)--> Na₂O₂ (黄色火焰)
      - **与H₂O**
        * 2Na + 2H₂O → 2NaOH + H₂↑
        * 现象：浮、熔、游、响、红
      - **与酸** (剧烈，不宜直接实验)
        * 2Na + 2HCl → 2NaCl + H₂↑
    * **保存**
      * 煤油/石蜡 (隔绝空气和水)
