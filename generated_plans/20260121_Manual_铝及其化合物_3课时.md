# 铝及其化合物

> 🤖 模型：gemini-flash-latest | 📅 时间：2026-01-21 14:19 | 🕒 课时：3 (每课时40min)

## 环节一：学习目标

本课题《铝及其化合物》旨在通过对铝单质、氧化铝和氢氧化铝性质的系统学习，使学生建立对两性物质的认知模型，提升实验探究能力和化学计算素养。

1.  **知识理解与模型认知**：理解铝单质、氧化铝（Al₂O₃）和氢氧化铝（Al(OH)₃）的物理性质和化学性质，特别是掌握其**两性**特征，并能书写相关的化学方程式和离子方程式。
2.  **实验探究与证据推理**：通过设计和观察 Al(OH)₃ 的制备及其与强酸、强碱反应的实验，培养学生基于实验现象进行证据推理的能力，并掌握检验 Al³⁺ 和 [Al(OH)₄]⁻ 离子的方法。
3.  **科学素养与化学计算**：掌握 Al³⁺ 和 [Al(OH)₄]⁻ 相互转化的定量关系，能运用守恒思想和离子方程式进行相关化学计算，提升分析和解决实际问题的能力（如溶液中离子的分离与提纯）。

## 环节二：情景创设

**情景选择：生活情境**

**情景描述：**
教师展示生活中常见的铝制品（如铝合金门窗、易拉罐、铝箔），并提出疑问：
“我们知道铝是一种非常活泼的金属，在金属活动性顺序中排在氢之前，理论上它应该很容易被腐蚀。但为什么我们日常使用的铝制品却具有优异的抗腐蚀性？此外，胃药中常含有氢氧化铝（Al(OH)₃），它能中和胃酸（HCl），但如果胃酸过多，它会不会又被溶解掉呢？”

**设计意图：**
通过生活中的矛盾现象（活泼金属却耐腐蚀；作为碱性物质却能溶解在过量的碱中），引发学生的认知冲突。将学习重点导向铝及其化合物的**两性**这一核心概念，明确本课题的学习目标——探究铝的“双重性格”。（时间控制：4分钟）

本课题共分为 3 个课时，每课时 40 分钟。

### 第1课时：铝单质的性质与应用

**子主题**：铝的物理性质、氧化膜和与酸碱的反应。

#### 任务 1.1：铝的物理性质与氧化膜 (15 分钟)

1.  **自主学习**：
    *   **学习单问题**：查阅资料，总结铝单质的主要物理性质（颜色、密度、导电性、延展性）。
    *   **核心问题**：铝的化学性质活泼，但为何铝制品耐腐蚀？请写出铝在空气中发生反应的化学方程式。

2.  **归纳小结**：
    *   **物理性质**：银白色、密度小（轻）、导电导热性好、延展性好。
    *   **化学性质（钝化）**：铝易与空气中的 O₂ 反应，生成一层致密、坚固的氧化铝薄膜（Al₂O₃），阻止内部铝进一步氧化。
    *   **方程式**：4Al + 3O₂ → 2Al₂O₃

3.  **评价训练**：
    *   **检测题 1-1**：下列关于铝的叙述中，不正确的是（ ）
        A. 铝是地壳中含量最多的金属元素。
        B. 铝制品在空气中不易被腐蚀，是因为铝的化学性质不活泼。
        C. 铝是电的良导体，可用于制作电线。
        D. 铝合金具有密度小、强度高的特点，可用于制造飞机。
    *   *答案：B (应是化学性质活泼，但表面有致密氧化膜保护)*

#### 任务 1.2：铝与酸和碱的反应 (25 分钟)

1.  **合作探究**：
    *   **实验探究**：将打磨后的铝片分别投入稀硫酸（H₂SO₄）溶液和氢氧化钠（NaOH）溶液中，观察现象并比较反应速率。
    *   **讨论问题**：铝与酸反应是正常的金属性质，但它为什么能与强碱溶液反应？请尝试写出铝与 NaOH 溶液反应的离子方程式。

2.  **归纳小结**：
    *   **与酸反应**：2Al + 6H⁺ → 2Al³⁺ + 3H₂↑
    *   **与碱反应**：铝单质具有两性特征。
        *   2Al + 2OH⁻ + 6H₂O → 2[Al(OH)₄]⁻ + 3H₂↑
    *   **结论**：铝既能与强酸反应放出 H₂，也能与强碱反应放出 H₂，体现了其特殊的化学活泼性。

3.  **评价训练**：
    *   **检测题 1-2**：将一小块打磨后的铝片投入到足量的 NaOH 溶液中，观察到有气泡产生。请写出该反应的离子方程式。
    *   *答案：2Al + 2OH⁻ + 6H₂O → 2[Al(OH)₄]⁻ + 3H₂↑*

### 第2课时：氧化铝和氢氧化铝的两性

**子主题**：Al₂O₃ 和 Al(OH)₃ 的两性本质。

#### 任务 2.1：氧化铝的两性 (15 分钟)

1.  **自主学习**：
    *   **学习单问题**：根据 Al₂O₃ 的结构和性质，判断它是酸性氧化物、碱性氧化物还是两性氧化物？
    *   **核心任务**：写出 Al₂O₃ 分别与盐酸（HCl）和氢氧化钠（NaOH）溶液反应的离子方程式。

2.  **归纳小结**：
    *   **性质**：Al₂O₃ 是典型的两性氧化物。
    *   **与酸反应（显碱性）**：Al₂O₃ + 6H⁺ → 2Al³⁺ + 3H₂O
    *   **与碱反应（显酸性）**：Al₂O₃ + 2OH⁻ + 3H₂O → 2[Al(OH)₄]⁻
    *   **应用**：Al₂O₃ 熔点高，可用于制作耐火材料；天然的 Al₂O₃ 晶体（如红宝石、蓝宝石）是宝石的主要成分。

3.  **评价训练**：
    *   **检测题 2-1**：下列物质中，不能与 Al₂O₃ 反应的是（ ）
        A. 稀 HNO₃
        B. 浓 NaOH 溶液
        C. H₂O
        D. 熔融的 Na₂CO₃
    *   *答案：C (Al₂O₃ 不溶于水)*

#### 任务 2.2：氢氧化铝的两性 (25 分钟)

1.  **合作探究**：
    *   **实验探究**：
        1.  制备 Al(OH)₃：向 AlCl₃ 溶液中滴加少量氨水（NH₃·H₂O），观察现象。
        2.  性质验证：将制得的 Al(OH)₃ 沉淀分别加入过量稀 HCl 和过量 NaOH 溶液中，观察现象。
    *   **讨论问题**：Al(OH)₃ 沉淀在两种溶液中都溶解了，这说明了什么？请写出 Al(OH)₃ 溶解在 NaOH 溶液中的离子方程式。

2.  **归纳小结**：
    *   **制备**：Al³⁺ + 3NH₃·H₂O → Al(OH)₃↓ + 3NH₄⁺ (注意：强碱过量会溶解沉淀，故常用弱碱制备)
    *   **两性**：Al(OH)₃ 是两性氢氧化物。
        *   **与酸反应（显碱性）**：Al(OH)₃ + 3H⁺ → Al³⁺ + 3H₂O
        *   **与碱反应（显酸性）**：Al(OH)₃ + OH⁻ → [Al(OH)₄]⁻
    *   **热稳定性**：2Al(OH)₃ $\xrightarrow{\Delta}$ Al₂O₃ + 3H₂O

3.  **评价训练**：
    *   **检测题 2-2**：写出 Al(OH)₃ 作为胃药中和胃酸（HCl）的离子方程式，并解释为什么服用过量胃药不会导致胃部碱性过高。
    *   *答案：Al(OH)₃ + 3H⁺ → Al³⁺ + 3H₂O。因为 Al(OH)₃ 难溶于水，只有在酸性条件下才反应，不会使胃液过度碱化。*

### 第3课时：铝盐与偏铝酸盐的转化

**子主题**：Al³⁺ 和 [Al(OH)₄]⁻ 的相互转化与定量计算。

#### 任务 3.1：铝盐与偏铝酸盐的相互转化 (25 分钟)

1.  **合作探究**：
    *   **情境分析**：向 AlCl₃ 溶液中滴加 NaOH 溶液，现象是先产生沉淀，后沉淀溶解。向 Na[Al(OH)₄] 溶液中滴加 HCl 溶液，现象是先产生沉淀，后沉淀溶解。
    *   **讨论问题**：
        1.  写出 Al³⁺ 转化为 [Al(OH)₄]⁻ 的两个阶段的离子方程式。
        2.  如何利用酸或碱，将 Al³⁺ 和 Fe³⁺ 两种离子分离？

2.  **归纳小结**：
    *   **Al³⁺ → [Al(OH)₄]⁻**：
        *   沉淀：Al³⁺ + 3OH⁻ → Al(OH)₃↓
        *   溶解：Al(OH)₃ + OH⁻ → [Al(OH)₄]⁻
    *   **[Al(OH)₄]⁻ → Al³⁺**：
        *   沉淀：[Al(OH)₄]⁻ + H⁺ → Al(OH)₃↓ + H₂O
        *   溶解：Al(OH)₃ + 3H⁺ → Al³⁺ + 3H₂O
    *   **分离提纯**：利用 Al(OH)₃ 的两性，加入过量强碱（如 NaOH）可将 Al³⁺ 转化为可溶性的 [Al(OH)₄]⁻，而 Fe³⁺ 转化为难溶的 Fe(OH)₃，实现分离。

3.  **评价训练**：
    *   **检测题 3-1**：向含有 1 mol AlCl₃ 的溶液中缓慢滴加 2 mol NaOH 溶液，此时溶液中 Al³⁺ 物质的量浓度如何变化？写出此时发生的主要反应的离子方程式。
    *   *答案：Al³⁺ 浓度降低。反应方程式：Al³⁺ + 3OH⁻ → Al(OH)₃↓。由于 OH⁻ 不足（需 3 mol），沉淀未完全，但未发生溶解。*

#### 任务 3.2：铝盐的应用 (10 分钟)

1.  **自主学习**：
    *   **学习单问题**：明矾（KAl(SO₄)₂·12H₂O）在水处理中起什么作用？写出 Al³⁺ 在水中发生水解的离子方程式。

2.  **归纳小结**：
    *   **净水原理**：Al³⁺ 发生水解反应：Al³⁺ + 3H₂O ⇌ Al(OH)₃ (胶体) + 3H⁺。生成的 Al(OH)₃ 胶体具有巨大的表面积，能吸附水中的悬浮物和杂质，达到净水目的。

3.  **评价训练**：
    *   **检测题 3-2**：解释为什么用明矾净水时，水的 pH 会略微下降。
    *   *答案：因为 Al³⁺ 水解生成 Al(OH)₃ 胶体的同时，也生成了 H⁺ 离子，导致溶液酸性增强，pH 下降。*

**总结要点**：
1.  **金属两性**：Al 既能与强酸反应，也能与强碱反应。
2.  **化合物两性**：Al₂O₃ 和 Al(OH)₃ 既能与酸反应，也能与碱反应。
3.  **转化关系**：Al³⁺ 和 [Al(OH)₄]⁻ 可以通过控制酸碱的量实现相互转化，其桥梁是 Al(OH)₃。

(1) 除去铜粉中混有的少量铝粉（使用 NaOH 溶液）：
(2) 向 Na[Al(OH)₄] 溶液中通入过量 CO₂ 气体：

**3. 综合计算与分析 (转化关系)**

向 100 mL 0.1 mol·L⁻¹ 的 AlCl₃ 溶液中，逐滴加入 0.1 mol·L⁻¹ 的 NaOH 溶液。
(1) 当加入 300 mL NaOH 溶液时，溶液中沉淀的物质的量是多少？
(2) 若要使沉淀完全溶解，至少还需要加入多少毫升 NaOH 溶液？

---
**参考答案：**

**1. 选择题**
*   **答案：B**
    *   A 选项中，K⁺、Na⁺、Cl⁻ 均可共存，但 Al³⁺ 和 [Al(OH)₄]⁻ 会发生 Al³⁺ + [Al(OH)₄]⁻ → 2Al(OH)₃↓ 反应，不能共存。
    *   C 选项中，OH⁻ 和 Al³⁺ 会反应生成 Al(OH)₃↓，不能共存。
    *   D 选项中，Al³⁺ 和 HCO₃⁻ 会发生双水解反应：Al³⁺ + 3HCO₃⁻ → Al(OH)₃↓ + 3CO₂↑，不能共存。

**2. 离子方程式书写**
(1) 2Al + 2OH⁻ + 6H₂O → 2[Al(OH)₄]⁻ + 3H₂↑
(2) [Al(OH)₄]⁻ + CO₂ → Al(OH)₃↓ + HCO₃⁻

**3. 综合计算与分析**
*   原始 Al³⁺ 物质的量 $n(\text{Al}^{3+}) = 0.1 \text{ L} \times 0.1 \text{ mol}\cdot\text{L}^{-1} = 0.01 \text{ mol}$。
*   (1) 加入 OH⁻ 物质的量 $n(\text{OH}^{-}) = 0.3 \text{ L} \times 0.1 \text{ mol}\cdot\text{L}^{-1} = 0.03 \text{ mol}$。
    *   沉淀反应：$\text{Al}^{3+} + 3\text{OH}^{-} \rightarrow \text{Al}(\text{OH})_{3}\downarrow$
    *   根据化学计量关系，完全沉淀需要 $0.01 \text{ mol} \times 3 = 0.03 \text{ mol}$ 的 $\text{OH}^{-}$。
    *   恰好完全沉淀。沉淀物质的量为 $0.01 \text{ mol}$。
*   (2) 沉淀溶解反应：$\text{Al}(\text{OH})_{3} + \text{OH}^{-} \rightarrow [\text{Al}(\text{OH})_{4}]^{-}$
    *   溶解 $0.01 \text{ mol}$ 的 $\text{Al}(\text{OH})_{3}$ 还需要 $0.01 \text{ mol}$ 的 $\text{OH}^{-}$。
    *   所需 $\text{NaOH}$ 溶液体积 $V = \frac{0.01 \text{ mol}}{0.1 \text{ mol}\cdot\text{L}^{-1}} = 0.1 \text{ L} = 100 \text{ mL}$。
    *   **答案：** (1) $0.01 \text{ mol}$； (2) $100 \text{ mL}$。
