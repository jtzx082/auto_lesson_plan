### 任务一：电子排布的“住房”规则——三大原理

**1. 自主学习**

*   **泡利不相容原理**：一个原子轨道中最多能容纳多少个电子？这些电子的自旋方向有什么要求？
*   **能量最低原理**：电子填充轨道的顺序是什么？请写出前四能层的轨道填充顺序（如 1s, 2s...）。
*   **洪特规则**：当电子填充能量相同的简并轨道（如 2p 轨道）时，电子的填充方式是怎样的？

**2. 归纳小结**

教师引导学生总结：
*   **轨道容量**：s轨道最多容纳2个，p轨道最多容纳6个，d轨道最多容纳10个。
*   **填充顺序**：严格遵循能量最低原理，电子总是优先占据能量最低的轨道（1s < 2s < 2p < 3s < 3p < 4s < 3d < ...）。
*   **洪特规则**：在能量相同的轨道上，电子总是优先单独占据，且自旋方向相同。

### 任务三：特殊稳定性与异常排布

**1. 合作探究**

**探究问题：**
根据能量最低原理，铬（Cr，Z=24）的电子排布式应为 [Ar] 3d⁴ 4s²。但实验测得其基态排布式为 [Ar] 3d⁵ 4s¹。
*   讨论：为什么 Cr 会出现这种“异常”排布？
*   讨论：请总结哪些电子排布状态具有特殊的稳定性？（提示：关注 d 轨道）

**2. 归纳小结**

*   **特殊稳定性**：当原子轨道处于**全充满**（如 d¹⁰, p⁶）、**半充满**（如 d⁵, p³）或**全空**（如 d⁰, p⁰）状态时，原子体系的能量较低，结构更加稳定。
*   **异常排布**：为了达到更高的稳定性，少数元素（如 Cr, Cu）会发生电子“跃迁”，从能量较高的 s 轨道转移到能量稍低的 d 轨道，形成半充满或全充满结构。
    *   Cu (Z=29)：[Ar] 3d¹⁰ 4s¹ (而非 3d⁹ 4s²)

| **电子排布** | **能量最低原理** (决定填充顺序) | **电子排布式** (1s² 2s²...) | **特殊稳定性** (d⁵, d¹⁰) |
| | **泡利不相容原理** (决定轨道容量) | **电子排布图** (箭头表示自旋) | **元素性质** (决定价电子数) |
| | **洪特规则** (决定简并轨道填充) | **简化排布式** ([Ne] 3s²...) | **异常排布** (Cr, Cu) |
