### 【情境引入】 (5分钟)

**教师活动：** (展示实验或图片) 准备一试管稀硫酸酸化的 $\text{KMnO}_4$ 溶液（紫色），然后滴加 $\text{FeSO}_4$ 溶液。

**师：** 同学们观察，紫色的高锰酸钾溶液发生了什么变化？（颜色迅速褪去，变为浅黄色/无色）。这种剧烈的颜色变化背后，是电子的快速转移。我们知道，这是一个氧化还原反应：$\text{Fe}^{2+}$ 被氧化成 $\text{Fe}^{3+}$，$\text{MnO}_4^{-}$ 被还原成 $\text{Mn}^{2+}$。

**师：** 如果我们用传统的化合价升降法来配平这个反应，步骤相对简单。但对于一些复杂的、涉及多氧酸根的离子反应，传统方法往往显得繁琐，甚至难以平衡 $\text{H}$ 和 $\text{O}$ 原子。今天，我们将学习一种更系统、更严谨的方法——**离子-电子法 (半反应法)**，它是电化学研究的基础。

**板书：** 课题：氧化还原反应的配平 (离子-电子法)

### 【教学过程】 (35分钟)

#### 环节一：方法介绍与基本步骤梳理 (5分钟)

**师：** 离子-电子法基于将总反应拆分成两个独立的半反应：氧化半反应（失电子）和还原半反应（得电子）。

| 步骤 | 操作内容 | 目的 |
| :--- | :--- | :--- |
| **1. 拆分** | 将总反应拆分为氧化半反应和还原半反应。 | 明确得失电子主体。 |
| **2. 平衡主体** | 配平除 $\text{H}$ 和 $\text{O}$ 外的其他原子。 | 质量守恒（主体原子）。 |
| **3. 平衡 $\text{O}$** | 在缺 $\text{O}$ 的一侧，用 $\text{H}_2\text{O}$ 配平氧原子。 | 质量守恒（氧原子）。 |
| **4. 平衡 $\text{H}$** | 在缺 $\text{H}$ 的一侧，用 $\text{H}^{+}$ 配平氢原子 (酸性介质)。 | 质量守恒（氢原子）。 |
| **5. 平衡电荷** | 在电荷高的一侧，加入 $e^{-}$ 配平电荷。 | 电荷守恒。 |
| **6. 合并** | 使两个半反应的 $e^{-}$ 数相等，然后相加抵消 $e^{-}$。 | 电子得失守恒。 |
| **7. 检查** | 检查原子守恒和电荷守恒。 | 确保配平正确。 |

#### 环节二：例题精讲与师生协作 (15分钟)

**例题一 (基础)：** 配平酸性介质下的反应：
$$\text{Fe}^{2+} + \text{MnO}_4^{-} \longrightarrow \text{Fe}^{3+} + \text{Mn}^{2+}$$

**师生互动：**

| 步骤 | 氧化半反应 (师导) | 还原半反应 (生答) |
| :--- | :--- | :--- |
| **1. 拆分** | $\text{Fe}^{2+} \longrightarrow \text{Fe}^{3+}$ | $\text{MnO}_4^{-} \longrightarrow \text{Mn}^{2+}$ |
| **2. 平衡主体** | $\text{Fe}^{2+} \longrightarrow \text{Fe}^{3+}$ (已平) | $\text{MnO}_4^{-} \longrightarrow \text{Mn}^{2+}$ (已平) |
| **3. 平衡 $\text{O}$** | 无 $\text{O}$ | $\text{MnO}_4^{-} \longrightarrow \text{Mn}^{2+} + 4\text{H}_2\text{O}$ |
| **4. 平衡 $\text{H}$** | 无 $\text{H}$ | $8\text{H}^{+} + \text{MnO}_4^{-} \longrightarrow \text{Mn}^{2+} + 4\text{H}_2\text{O}$ |
| **5. 平衡电荷** | $\text{Fe}^{2+} \longrightarrow \text{Fe}^{3+} + 1e^{-}$ (失1e) | $8\text{H}^{+} + \text{MnO}_4^{-} + 5e^{-} \longrightarrow \text{Mn}^{2+} + 4\text{H}_2\text{O}$ (得5e) |
| **6. 合并** | 氧化半反应 ($\times 5$)：$5\text{Fe}^{2+} \longrightarrow 5\text{Fe}^{3+} + 5e^{-}$ | 还原半反应 ($\times 1$)：$8\text{H}^{+} + \text{MnO}_4^{-} + 5e^{-} \longrightarrow \text{Mn}^{2+} + 4\text{H}_2\text{O}$ |
| **总反应** | **$5\text{Fe}^{2+} + \text{MnO}_4^{-} + 8\text{H}^{+} \longrightarrow 5\text{Fe}^{3+} + \text{Mn}^{2+} + 4\text{H}_2\text{O}$** | |

**师：** **(检查环节)** 左边电荷：$5 \times (+2) + (-1) + 8 \times (+1) = +17$。右边电荷：$5 \times (+3) + (+2) = +17$。电荷守恒，配平成功！

#### 环节三：独立尝试与难点突破 (15分钟)

**例题二 (挑战)：** 配平酸性介质下的反应：
$$\text{Cr}_2\text{O}_7^{2-} + \text{SO}_3^{2-} \longrightarrow \text{Cr}^{3+} + \text{SO}_4^{2-}$$

**(学生先尝试拆分和平衡主体原子，教师巡视指导)**

**师：** 注意，在还原剂 $\text{Cr}_2\text{O}_7^{2-}$ 中，铬原子是两个，必须先平衡主体！

| 步骤 | 还原半反应 | 氧化半反应 |
| :--- | :--- | :--- |
| **1. 拆分** | $\text{Cr}_2\text{O}_7^{2-} \longrightarrow \text{Cr}^{3+}$ | $\text{SO}_3^{2-} \longrightarrow \text{SO}_4^{2-}$ |
| **2. 平衡主体** | $\text{Cr}_2\text{O}_7^{2-} \longrightarrow 2\text{Cr}^{3+}$ | $\text{SO}_3^{2-} \longrightarrow \text{SO}_4^{2-}$ (已平) |
| **3. 平衡 $\text{O}$** | $\text{Cr}_2\text{O}_7^{2-} \longrightarrow 2\text{Cr}^{3+} + 7\text{H}_2\text{O}$ | $\text{H}_2\text{O} + \text{SO}_3^{2-} \longrightarrow \text{SO}_4^{2-}$ |
| **4. 平衡 $\text{H}$** | $14\text{H}^{+} + \text{Cr}_2\text{O}_7^{2-} \longrightarrow 2\text{Cr}^{3+} + 7\text{H}_2\text{O}$ | $\text{H}_2\text{O} + \text{SO}_3^{2-} \longrightarrow \text{SO}_4^{2-} + 2\text{H}^{+}$ |
| **5. 平衡电荷** | $14\text{H}^{+} + \text{Cr}_2\text{O}_7^{2-} + 6e^{-} \longrightarrow 2\text{Cr}^{3+} + 7\text{H}_2\text{O}$ (得6e) | $\text{H}_2\text{O} + \text{SO}_3^{2-} \longrightarrow \text{SO}_4^{2-} + 2\text{H}^{+} + 2e^{-}$ (失2e) |
| **6. 合并** | 还原半反应 ($\times 1$) | 氧化半反应 ($\times 3$)：$3\text{H}_2\text{O} + 3\text{SO}_3^{2-} \longrightarrow 3\text{SO}_4^{2-} + 6\text{H}^{+} + 6e^{-}$ |
| **7. 整理** | $14\text{H}^{+} + 3\text{H}_2\text{O} + \text{Cr}_2\text{O}_7^{2-} + 3\text{SO}_3^{2-} \longrightarrow 2\text{Cr}^{3+} + 7\text{H}_2\text{O} + 3\text{SO}_4^{2-} + 6\text{H}^{+}$ | |

**总反应 (约简 $\text{H}_2\text{O}$ 和 $\text{H}^{+}$):**
$$8\text{H}^{+} + \text{Cr}_2\text{O}_7^{2-} + 3\text{SO}_3^{2-} \longrightarrow 2\text{Cr}^{3+} + 4\text{H}_2\text{O} + 3\text{SO}_4^{2-}$$

**师：** **(检查环节)** 左边电荷：$8 \times (+1) + (-2) + 3 \times (-2) = 8 - 2 - 6 = 0$。右边电荷：$2 \times (+3) + 3 \times (-2) = 6 - 6 = 0$。电荷守恒，配平成功！

#### 环节四：总结升华 (5分钟)

**师：** 离子-电子法之所以严谨，是因为它同时满足了三项重要的化学守恒定律：
1. **原子守恒** (步骤2-4)
2. **电荷守恒** (步骤5)
3. **电子守恒** (步骤6)

在酸性介质中，我们使用 $\text{H}_2\text{O}$ 和 $\text{H}^{+}$ 来配平 $\text{O}$ 和 $\text{H}$。如果是在碱性介质中，我们如何处理？（启发学生思考：用 $\text{OH}^{-}$ 代替 $\text{H}^{+}$ 来平衡电荷，但平衡 $\text{O}$ 和 $\text{H}$ 的基本思路不变，只是需要用 $\text{OH}^{-}$ 和 $\text{H}_2\text{O}$ 组合。）

### 【课后作业】

1. **巩固练习 (酸性介质)：**
   请用离子-电子法配平下列反应，并指出氧化剂和还原剂：
   $$\text{I}^{-} + \text{NO}_3^{-} \longrightarrow \text{I}_2 + \text{NO} \quad (\text{H}^{+} \text{参与})$$

2. **拓展思考 (碱性介质尝试)：**
   请尝试配平碱性条件下的反应：
   $$\text{Br}_2 \longrightarrow \text{BrO}_3^{-} + \text{Br}^{-} \quad (\text{OH}^{-} \text{参与})$$
   *提示：在碱性条件下，每加 $1\text{H}^{+}$ 平衡 $\text{H}$，同时在两边各加 $1\text{OH}^{-}$，将 $\text{H}^{+}$ 和 $\text{OH}^{-}$ 组合成 $\text{H}_2\text{O}$。*

3. **概念辨析：**
   在配平氧化还原反应时，为什么我们必须先平衡原子，最后才平衡电荷？（从守恒的角度解释。）
