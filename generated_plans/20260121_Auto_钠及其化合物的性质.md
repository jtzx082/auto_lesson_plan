# 钠及其化合物的性质

> 🤖 模型：gemini-2.5-flash | 📅 时间：2026-01-21 14:06

## 环节一：学习目标

1.  通过观察实验现象、阅读教材资料，了解钠的物理性质，探究并掌握钠与水反应的现象、产物和化学方程式，认识钠的活泼性，培养实验观察与分析能力。
2.  结合生活情境（如呼吸面具），理解过氧化钠（Na₂O₂）的特殊化学性质（与水、CO₂反应），能正确书写相关化学方程式，并解释其在实际中的应用，提升分析问题和解决问题的能力。
3.  初步认识氢氧化钠（NaOH）、碳酸钠（Na₂CO₃）和碳酸氢钠（NaHCO₃）等钠的重要化合物的俗名、主要性质和用途，形成对物质性质与用途关系的初步认识。

## 环节二：情景创设

**情景选择：** 实验情境

**具体设计：**
在课堂开始时，教师播放一段钠与水反应的实验视频（或进行演示实验，注意安全）。视频中，一小块金属钠投入水中，迅速熔成小球，在水面浮动，发出“嘶嘶”声，并有气体放出，同时小球逐渐变小直至消失，滴入酚酞后溶液变红。

**设计意图：**
通过直观、生动的实验现象，给学生带来强烈的视觉冲击，激发学生对未知事物的好奇心和探究欲望。引导学生思考：这块“神奇”的金属是什么？它为什么会这样反应？反应后生成了什么？从而引发认知冲突，明确本节课将要探究的核心内容——钠及其化合物的性质，为后续的学习奠定基础。

## 环节三：任务驱动教学

### 任务一：活泼的金属钠

**目标：** 了解钠的物理性质，掌握钠与水反应的现象、产物和化学方程式。

1.  **合作探究**
    *   **问题1：** 结合刚才的实验现象，你观察到了钠的哪些物理性质？（如颜色、状态、硬度、密度、熔点等）
    *   **问题2：** 钠与水反应时，你观察到哪些现象？这些现象分别说明了什么？（如“浮”、“熔”、“游”、“响”、“红”）
    *   **问题3：** 钠与水反应生成了什么物质？请写出该反应的化学方程式，并指出反应中钠元素的化合价变化。
    *   **问题4：** 结合钠的活泼性，思考金属钠应该如何保存？

2.  **归纳小结**
    *   **物理性质：** 银白色金属，质软（可用小刀切割），密度比水小，熔点低（97.8℃），是电和热的良导体。
    *   **化学性质：**
        *   与非金属反应：
            *   与O₂反应：常温下与O₂反应生成氧化钠（Na₂O）；加热时剧烈燃烧生成过氧化钠（Na₂O₂）。
                4Na + O₂ → 2Na₂O (常温)
                2Na + O₂ $\xrightarrow{\Delta}$ Na₂O₂ (加热)
            *   与Cl₂反应：2Na + Cl₂ $\xrightarrow{点燃}$ 2NaCl
        *   与水反应：2Na + 2H₂O → 2NaOH + H₂↑
            *   **现象：** 浮（密度比水小）、熔（熔点低，反应放热）、游（H₂推动）、响（H₂燃烧）、红（生成NaOH使酚酞变红）。
    *   **保存：** 钠的化学性质非常活泼，常温下能与空气中的O₂和水反应，因此通常保存在煤油或石蜡油中，隔绝空气和水。

3.  **评价训练**
    1.  下列关于金属钠的叙述，正确的是（ ）
        A. 钠是银白色金属，熔点很高
        B. 钠的密度比水大，所以浮在水面上
        C. 钠与水反应时，生成H₂和NaOH
        D. 钠可以保存在酒精中
    2.  将一小块金属钠投入盛有酚酞试液的水中，观察到的现象有：①钠浮在水面上；②钠熔成小球；③小球在水面四处游动；④发出“嘶嘶”的响声；⑤溶液变红色。请写出钠与水反应的化学方程式：____________________。

### 任务二：奇特的过氧化钠

**目标：** 理解过氧化钠（Na₂O₂）的特殊性质（与水、CO₂反应），并能解释其在呼吸面具中的应用。

1.  **自主学习**
    *   **问题1：** 阅读教材中关于过氧化钠（Na₂O₂）的描述，了解其颜色、状态。
    *   **问题2：** 观看过氧化钠与水反应的视频，记录现象，并尝试写出化学方程式。该反应有何特点？（提示：产物中是否有气体？）
    *   **问题3：** 观看过氧化钠与二氧化碳（CO₂）反应的视频，记录现象，并尝试写出化学方程式。该反应有何特点？
    *   **问题4：** 结合过氧化钠与水和CO₂的反应，解释为什么过氧化钠可以作为呼吸面具和潜艇中的供氧剂？

2.  **归纳小结**
    *   **物理性质：** 淡黄色固体，粉末状。
    *   **化学性质：**
        *   与水反应：2Na₂O₂ + 2H₂O → 4NaOH + O₂↑
            *   **特点：** 反应剧烈，放热，有O₂生成。
        *   与CO₂反应：2Na₂O₂ + 2CO₂ → 2Na₂CO₃ + O₂↑
            *   **特点：** 反应有O₂生成，吸收CO₂。
    *   **用途：** 呼吸面具和潜艇中的供氧剂（吸收CO₂，放出O₂），漂白剂（强氧化性）。
    *   **Na₂O与Na₂O₂的区别：**
        *   颜色：Na₂O（白色），Na₂O₂（淡黄色）
        *   与水反应：Na₂O + H₂O → 2NaOH (无O₂放出)；2Na₂O₂ + 2H₂O → 4NaOH + O₂↑ (有O₂放出)
        *   与CO₂反应：Na₂O + CO₂ → Na₂CO₃ (无O₂放出)；2Na₂O₂ + 2CO₂ → 2Na₂CO₃ + O₂↑ (有O₂放出)

3.  **评价训练**
    1.  下列关于Na₂O₂的叙述中，正确的是（ ）
        A. Na₂O₂是白色固体
        B. Na₂O₂与水反应只生成NaOH
        C. Na₂O₂可用于呼吸面具中提供O₂
        D. Na₂O₂与CO₂反应生成Na₂CO₃和H₂O
    2.  写出过氧化钠与二氧化碳反应的化学方程式：____________________。该反应中，O₂⁻的化合价变化是____________________。
