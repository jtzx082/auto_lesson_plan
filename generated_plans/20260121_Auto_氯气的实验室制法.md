# 第1课时：氯气的实验室制备原理、装置与操作

## 环节一：学习目标

1.  了解氯气实验室制法的反应原理，能从氧化还原角度分析反应过程。
2.  掌握实验室制备氯气的发生、除杂、收集和尾气处理装置，并能正确连接和操作。
3.  理解氯气制备实验中的安全注意事项和环保要求，并能初步运用绿色化学理念评价实验方案。
4.  通过实验方案设计与分析，提升科学探究能力和解决实际问题的能力。

## 环节二：情景创设 (3分钟)

**教师导入**：同学们，我们知道氯气是一种重要的化工原料，广泛应用于自来水消毒、农药、塑料等生产。在工业上，氯气主要通过电解饱和食盐水的方法大规模生产。然而，在实验室中，我们通常无法进行电解实验来制备氯气，而且工业生产的条件和规模也与实验室要求大相径庭。

**提出问题，引发认知冲突**：那么，在实验室中，我们应该选择什么样的方法来制备氯气呢？这种方法需要满足哪些条件？氯气是一种有毒气体，我们又该如何安全、环保地进行实验呢？今天，我们就一起来探索氯气的实验室制法。

## 环节三：任务驱动教学

### 任务一：探究氯气的实验室制备原理

**1. [合作探究]**
**教师引导**：氯气 (Cl₂) 是一种单质，在化合物中氯元素通常显负价 (如 HCl 中的 Cl⁻)。要制备 Cl₂，需要将 Cl⁻ 氧化为 Cl₂。请同学们结合所学知识，讨论选择何种氧化剂以及反应条件。
*   **学生讨论**：
    *   需要一种氧化性较强的物质作氧化剂。
    *   常见的氧化剂有 MnO₂, KMnO₄, KClO₃ 等。
    *   考虑到实验室的普遍性和安全性，MnO₂ 是一个常见的选择。
    *   MnO₂ 与盐酸反应，需要浓盐酸，因为稀盐酸氧化性太弱。
    *   反应通常需要加热才能进行。
*   **教师总结并板书反应方程式**：
    二氧化锰与浓盐酸在加热条件下反应生成氯气。
    **4HCl(浓) + MnO₂(s) $\xrightarrow{\Delta}$ MnCl₂(aq) + Cl₂(g) + 2H₂O(l)**
*   **分析氧化还原反应**：
    *   指出氧化剂 (MnO₂) 和还原剂 (HCl)。
    *   分析电子转移：Mn从+4价降到+2价，得到2e⁻；Cl从-1价升到0价，失去1e⁻ (每生成1分子Cl₂有2个Cl⁻失去2e⁻)。
    *   指出反应中HCl的作用：一部分作还原剂，一部分作酸。

**2. 归纳小结**
*   **反应原理**：利用强氧化剂 (如MnO₂) 将浓盐酸中的Cl⁻氧化为Cl₂。
*   **反应物**：浓盐酸 (提供Cl⁻和H⁺)、二氧化锰 (提供氧化剂)。
*   **反应条件**：加热。
*   **氧化还原分析**：MnO₂是氧化剂，HCl是还原剂兼酸。每生成1 mol Cl₂，有2 mol HCl被氧化，另有2 mol HCl作酸。

**3. 评价训练**
1.  写出实验室制备氯气的化学方程式，并用双线桥法标出电子转移的方向和数目。
    *   **参考答案**：
        ```
        4HCl(浓) + MnO₂(s) $\xrightarrow{\Delta}$ MnCl₂(aq) + Cl₂(g) + 2H₂O(l)
        ```
        (双线桥：从MnO₂的Mn到MnCl₂的Mn，得2e⁻；从4HCl中的2个Cl到Cl₂，失2e⁻)
2.  在上述反应中，若将浓盐酸改为稀盐酸，能否制得氯气？请说明理由。
    *   **参考答案**：不能。稀盐酸的还原性太弱，不足以被MnO₂氧化生成Cl₂。

### 任务二：设计氯气的实验室制备装置

**1. [合作探究]**
**教师引导**：氯气是一种黄绿色、有毒、密度比空气大、可溶于水 (但溶解度不大) 的气体。结合这些性质和制备原理 (固液加热)，请同学们分组讨论并设计一套完整的氯气制备、收集及尾气处理装置，并说明各部分的作用。
*   **学生讨论**：
    *   **发生装置**：固液加热，选择圆底烧瓶、分液漏斗、酒精灯、铁架台。
    *   **除杂装置**：
        *   除去挥发出的HCl气体：用饱和食盐水洗气瓶 (Cl₂在饱和食盐水中溶解度小，HCl溶解度大)。
        *   除去水蒸气：用浓硫酸洗气瓶 (浓硫酸有吸水性)。
    *   **收集装置**：氯气密度比空气大，用向上排空气法。
    *   **尾气处理装置**：氯气有毒，且能与碱反应，用NaOH溶液吸收 (2NaOH + Cl₂ → NaCl + NaClO + H₂O)。
*   **教师指导并绘制装置图**：
    *   **发生装置**：圆底烧瓶 (盛MnO₂)，分液漏斗 (滴加浓HCl)，导气管。
    *   **除杂装置**：洗气瓶A (饱和食盐水，长进短出)，洗气瓶B (浓H₂SO₄，长进短出)。
    *   **收集装置**：集气瓶 (向上排空气法)。
    *   **尾气处理装置**：烧杯或洗气瓶 (NaOH溶液，防倒吸)。
*   **强调连接顺序**：发生 → 除HCl → 除H₂O → 收集 → 尾气处理。

**2. 归纳小结**
*   **发生装置**：固液加热型，用于MnO₂与浓盐酸反应，如圆底烧瓶、分液漏斗、酒精灯等。
*   **除杂装置**：
    *   **饱和食盐水洗气瓶**：除去氯气中混有的氯化氢气体，利用Cl₂在饱和食盐水中溶解度小，HCl极易溶于水。
    *   **浓硫酸洗气瓶**：除去氯气中混有的水蒸气，利用浓硫酸的吸水性。
*   **收集装置**：向上排空气法，因Cl₂密度比空气大。
*   **尾气处理装置**：氢氧化钠溶液吸收多余的Cl₂，防止污染空气，并注意防倒吸。

**3. 评价训练**
1.  请绘制一套完整的实验室制备、收集干燥氯气并进行尾气处理的装置图，并标明各装置中盛放的试剂名称。
    *   **参考答案**：(学生绘制示意图，包含发生、除杂、收集、尾气处理，并标注MnO₂/浓HCl、饱和食盐水、浓H₂SO₄、NaOH溶液)
2.  若在除杂装置中，先通过浓硫酸再通过饱和食盐水，会有什么后果？
    *   **参考答案**：若先通过浓硫酸，再通过饱和食盐水，则氯气中的水蒸气会被饱和食盐水重新带入，导致收集到的氯气不干燥。因此，除杂顺序必须是先除HCl，后除水蒸气。

### 任务三：氯气制备的注意事项与绿色化学

**1. [自主学习]**
**教师引导**：请同学们阅读教材中关于氯气制备实验的注意事项，并思考如何从绿色化学的角度评价和改进该实验方案。
*   **学生阅读与思考**：
    *   浓盐酸具有挥发性和腐蚀性。
    *   氯气有毒，需要通风和尾气处理。
    *   加热温度不宜过高，防止副反应或试剂分解。
    *   防倒吸装置的重要性。
    *   反应需要加热，能耗较高。
    *   反应产物有MnCl₂，含有重金属离子。
    *   能否寻找更温和的反应条件或更环保的氧化剂？

**2. 归纳小结**
*   **实验操作注意事项**：
    *   **通风橱内操作**：氯气有毒，防止吸入。
    *   **浓盐酸的加入**：通过分液漏斗缓慢滴加，控制反应速率。
    *   **加热控制**：缓慢均匀加热，避免剧烈反应和浓盐酸过度挥发。
    *   **防倒吸**：尾气吸收装置应加防倒吸装置，防止NaOH溶液倒吸入收集瓶或反应瓶。
    *   **装置气密性**：实验前必须检查装置气密性。
*   **绿色化学理念的应用**：
    *   **原子经济性**：该反应中，HCl有部分未转化为Cl₂，原子利用率不高。
    *   **反应条件温和化**：MnO₂与浓HCl反应需要加热，能耗较高。
    *   **无害化**：反应产物MnCl₂含有重金属Mn²⁺，对环境有潜在污染。
    *   **改进方向**：寻找在常温下即可反应的氧化剂 (如KMnO₄、KClO₃等)，或开发更环保的催化剂。
*   **拓展**：介绍KMnO₄与浓盐酸反应制备Cl₂ (2KMnO₄ + 16HCl(浓) → 2KCl + 2MnCl₂ + 5Cl₂↑ + 8H₂O)，该反应常温即可进行，但KMnO₄价格较高，且氧化性过强，反应不易控制。

**3. 评价训练**
1.  实验室制备氯气时，在尾气处理装置中，除了用NaOH溶液吸收外，还需要特别注意什么？请解释原因。
    *   **参考答案**：还需要注意防止倒吸。因为氯气在NaOH溶液中的溶解度很大，若导管直接插入液面以下，当气体停止通入时，洗气瓶内压强骤降，NaOH溶液会倒吸入前面的装置，造成危险或污染。
2.  比较用MnO₂和KMnO₄制备氯气的两种方法，从反应条件和原子利用率角度分析，哪种方法更符合绿色化学理念？
    *   **参考答案**：
        *   **反应条件**：KMnO₄法常温即可反应，能耗低，更符合绿色化学理念。MnO₂法需要加热。
        *   **原子利用率**：两种方法都有部分HCl作酸未被氧化，原子利用率均不高。但KMnO₄法中，KMnO₄的Mn元素也转化为MnCl₂，与MnO₂法类似。综合来看，KMnO₄法在反应条件上更具优势。

    (2) 装置甲中盛放的试剂是\_\_\_\_\_\_\_\_\_\_，其作用是\_\_\_\_\_\_\_\_\_\_。
    (3) 装置乙中盛放的试剂是\_\_\_\_\_\_\_\_\_\_，其作用是\_\_\_\_\_\_\_\_\_\_。
    (4) 装置丙收集氯气的方法是\_\_\_\_\_\_\_\_\_\_，判断氯气是否收集满的实验方法是\_\_\_\_\_\_\_\_\_\_。

2.  某同学在实验室制备氯气时，发现收集到的氯气中含有少量氧气。他分析原因可能是：在加热过程中，二氧化锰与浓盐酸反应不充分，同时有少量空气进入了收集装置。为了避免氧气的混入，他提出在发生装置与除杂装置之间增加一个除氧装置。请你设计一个简单的除氧装置，并说明所用试剂和原理。

3.  已知氯气与水反应生成HCl和HClO (次氯酸)，次氯酸具有漂白性。为了验证干燥的氯气不具有漂白性，而湿润的氯气具有漂白性，某同学设计了以下实验方案：将干燥的氯气通入干燥的红色布条，布条不变色；然后将湿润的氯气通入干燥的红色布条，布条褪色。
    请你评价该实验方案是否严谨？若不严谨，请指出不足之处并提出改进建议。

---
**参考答案**

1.  (1) 4HCl(浓) + MnO₂(s) $\xrightarrow{\Delta}$ MnCl₂(aq) + Cl₂(g) + 2H₂O(l)
    (2) 饱和食盐水；除去氯气中混有的氯化氢气体。
    (3) 浓H₂SO₄；除去氯气中混有的水蒸气。
    (4) 向上排空气法；将湿润的KI-淀粉试纸放在集气瓶口，若试纸变蓝，则说明氯气已收集满。
    (5) NaOH溶液；吸收多余的氯气，防止污染空气。

2.  **设计方案**：可以在发生装置和除杂装置之间增加一个装有灼热铜网 (或铜粉) 的U形管。
    **所用试剂**：灼热的铜网 (或铜粉)。
    **原理**：氧气在加热条件下能与铜反应生成氧化铜 (2Cu + O₂ $\xrightarrow{\Delta}$ 2CuO)，从而除去氯气中的氧气。

3.  **评价**：该实验方案不严谨。
    **不足之处**：该方案只验证了“湿润的氯气具有漂白性”，但未能严谨地证明“干燥的氯气不具有漂白性”。因为“将干燥的氯气通入干燥的红色布条”后，布条不变色，但此时布条可能已经被氯气通入，其上的水分可能已经被氯气带走，导致布条实际处于干燥状态。当再通入湿润的氯气时，布条才褪色。这并不能排除干燥氯气本身具有漂白性的可能性（只是需要有水参与）。
    **改进建议**：
    *   **方案一**：将干燥的氯气分别通入干燥的红色布条和湿润的红色布条。若干燥布条不变色，湿润布条褪色，则能证明。
    *   **方案二**：将干燥的红色布条分成两部分，一部分直接放入干燥的氯气中，另一部分用蒸馏水润湿后放入干燥的氯气中。观察两部分布条的颜色变化。若干燥布条不变色，湿润布条褪色，则能证明。
