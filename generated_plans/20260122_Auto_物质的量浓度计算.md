## 环节二：情景创设 (3分钟)

**教师引导：** 
同学们，在初中我们学习了溶液的质量分数（ω），它表示溶质质量占溶液总质量的百分比。例如，医院常用的生理盐水是质量分数为0.9%的NaCl溶液。
现在请大家思考一个问题：如果实验室需要配制用于中和反应的NaOH溶液，或者需要计算某个化学反应中H₂SO₄的用量，我们是直接用质量分数来表示溶液的浓度更方便，还是用一种能直接反映溶液中溶质粒子数量多少的浓度表示方法更方便呢？为什么？

**学生思考与讨论：**
学生可能会回忆化学反应的本质是粒子间的相互作用，因此直接知道粒子数量会更方便。质量分数只涉及质量，还需要通过摩尔质量进行换算才能得到粒子数量。

**教师总结并引入：**
很好！正如大家所说，在化学实验和生产中，我们更关心的是单位体积溶液中含有多少“份”溶质粒子，也就是溶质的“物质的量”。为了方便化学计算和实验操作，化学上引入了一种新的浓度表示方法——**物质的量浓度**。今天，我们就来深入学习这一重要的概念和计算方法。

## 环节三：任务驱动教学

### 任务一：物质的量浓度的概念与物理意义

**目标：** 理解物质的量浓度的定义、符号、单位及其物理意义，并与质量分数进行区分。

**1. [自主学习] (8分钟)**
*   **阅读教材：** 学生阅读教材中关于“物质的量浓度”的部分，找出其定义、符号、单位。
*   **思考问题：**
    1.  什么是物质的量浓度？它的数学表达式是什么？
    2.  物质的量浓度的常用单位是什么？如何理解这个单位？
    3.  物质的量浓度与质量分数有什么异同？为什么化学上更常用物质的量浓度？
    4.  在表达式c = n/V中，n和V分别指什么？V的单位必须是什么？

**2. 归纳小结 (5分钟)**
*   **定义：** 物质的量浓度是指单位体积溶液中所含溶质B的物质的量。
*   **符号：** c_B 或 c
*   **表达式：** c_B = n_B / V_溶液
    *   n_B：溶质B的物质的量，单位：mol。
    *   V_溶液：溶液的体积，**注意单位必须是升 (L)**。
*   **单位：** mol/L 或 mol·L⁻¹。
*   **物理意义：** 例如，c(NaCl) = 0.1 mol/L，表示每升氯化钠溶液中含有0.1 mol的氯化钠。
*   **与质量分数的区别：** 物质的量浓度关注的是单位体积溶液中溶质的“粒子数量”，而质量分数关注的是溶质质量占溶液总质量的百分比。物质的量浓度更直接地反映了溶液中参与化学反应的粒子数目。

**3. 评价训练 (5分钟)**
1.  **判断题：**
    *   (1) 1 L水中溶解1 mol NaCl所得溶液的物质的量浓度为1 mol/L。( )
    *   (2) 从100 mL 1 mol/L NaOH溶液中取出10 mL，其物质的量浓度仍为1 mol/L。( )
    *   (3) 物质的量浓度为1 mol/L的NaCl溶液中含有1 mol NaCl。( )
    *   (4) 将20 g NaOH溶于水配成0.5 L溶液，则该溶液的物质的量浓度为1 mol/L。( )
    *   **答案：** (1) × (溶液体积不等于溶剂体积) (2) √ (浓度是均一的) (3) × (需指明溶液体积) (4) √ (n(NaOH)=20g/40g·mol⁻¹=0.5mol, c=0.5mol/0.5L=1mol/L)

### 任务二：物质的量浓度的计算与应用

**目标：** 掌握物质的量浓度的各种计算类型，包括由质量计算浓度、由浓度计算质量、以及溶液稀释的计算。

**1. [合作探究] (15分钟)**
*   **小组讨论与计算：** 教师提供以下计算题，学生以小组为单位进行讨论，并尝试完成计算。
    *   **例1 (直接计算)：** 将2 mol NaOH固体溶解在水中，配成500 mL溶液，求该溶液中NaOH的物质的量浓度。
        *   **提示：** 注意体积单位换算。
    *   **例2 (由浓度计算质量)：** 实验室需要配制250 mL 0.1 mol/L 的Na₂CO₃溶液，需要Na₂CO₃固体多少克？
        *   **提示：** 需要先计算溶质的物质的量，再计算质量。
    *   **例3 (含水合物的计算)：** 实验室配制500 mL 0.2 mol/L 的CuSO₄溶液，需要CuSO₄·5H₂O晶体多少克？(已知M(CuSO₄·5H₂O) = 250 g/mol)
        *   **提示：** CuSO₄·5H₂O中CuSO₄的物质的量与CuSO₄·5H₂O的物质的量相等。
    *   **例4 (溶液稀释)：** 将100 mL 2 mol/L 的HCl溶液稀释到200 mL，稀释后溶液的物质的量浓度是多少？
        *   **提示：** 稀释前后溶质的物质的量不变。

**2. 归纳小结 (5分钟)**
*   **基本公式：** c = n/V
*   **常用换算关系：**
    *   物质的量：n = m/M (m为质量，M为摩尔质量)
    *   溶质质量：m = n × M = c × V × M
    *   溶液体积：V = n/c
*   **溶液稀释定律：** 稀释前后溶质的物质的量不变，即 c₁V₁ = c₂V₂。
*   **计算注意事项：**
    *   V 始终是溶液的体积，单位为 L。
    *   注意单位的统一和换算。
    *   对于含结晶水的物质，计算时应使用其整个化学式的摩尔质量。

**3. 评价训练 (5分钟)**
1.  **计算题：** 某同学用4.9 g H₂SO₄配制成250 mL溶液，求该H₂SO₄溶液的物质的量浓度。(M(H₂SO₄) = 98 g/mol)
    *   **答案：** n(H₂SO₄) = 4.9 g / 98 g/mol = 0.05 mol。V = 250 mL = 0.25 L。c(H₂SO₄) = 0.05 mol / 0.25 L = 0.2 mol/L。
2.  **计算题：** 现有50 mL 0.5 mol/L 的KNO₃溶液，若要使溶液的物质的量浓度变为0.2 mol/L，需要加水多少毫升？
    *   **答案：** 设稀释后溶液体积为V₂。根据稀释定律 c₁V₁ = c₂V₂，则 0.5 mol/L × 50 mL = 0.2 mol/L × V₂。V₂ = (0.5 × 50) / 0.2 = 125 mL。需要加水体积 = V₂ - V₁ = 125 mL - 50 mL = 75 mL。

## 环节五：课堂检测 (7分钟)

1.  **基础题：**
    将23.4 g NaCl溶解在水中，配制成200 mL溶液。请计算该NaCl溶液的物质的量浓度。(M(NaCl) = 58.5 g/mol)
    *   **解析：** n(NaCl) = 23.4 g / 58.5 g/mol = 0.4 mol。V = 200 mL = 0.2 L。c(NaCl) = 0.4 mol / 0.2 L = 2 mol/L。
    *   **答案：** 2 mol/L

2.  **提升题一：**
    实验室现有100 mL 0.5 mol/L 的H₂SO₄溶液。若要将其稀释成0.1 mol/L的H₂SO₄溶液，需要加入多少毫升水？
    *   **解析：** 根据稀释定律 c₁V₁ = c₂V₂。0.5 mol/L × 100 mL = 0.1 mol/L × V₂。V₂ = (0.5 × 100) / 0.1 = 500 mL。需要加水体积 = V₂ - V₁ = 500 mL - 100 mL = 400 mL。
    *   **答案：** 400 mL

3.  **提升题二 (综合应用)：**
    某浓盐酸的质量分数为36.5%，密度为1.18 g/cm³，请计算该浓盐酸的物质的量浓度。(M(HCl) = 36.5 g/mol)
    *   **解析：** 假设有1 L (即1000 mL) 这种浓盐酸。
        *   溶液质量 m_溶液 = ρ × V_溶液 = 1.18 g/cm³ × 1000 cm³ = 1180 g。
        *   溶质HCl的质量 m_HCl = m_溶液 × ω = 1180 g × 36.5% = 1180 g × 0.365 = 430.7 g。
        *   溶质HCl的物质的量 n_HCl = m_HCl / M_HCl = 430.7 g / 36.5 g/mol ≈ 11.8 mol。
        *   物质的量浓度 c(HCl) = n_HCl / V_溶液 = 11.8 mol / 1 L = 11.8 mol/L。
        *   **或直接使用公式：** c = (1000ρω) / M = (1000 × 1.18 g/cm³ × 0.365) / 36.5 g/mol = 11.8 mol/L。
    *   **答案：** 11.8 mol/L
